## 2. 도입 배경 및 과제

### 기존 문제점

1. **카운슬링 혼잡**: 주말 오후 카운슬링 존에 대기 고객 집중, 상담 포기 후 이탈율 높음
2. **테스터 존 비효율**: 인기 테스터에 고객 집중, 나머지 테스터 방치 → 체험 만족도 저하
3. **상담원 미스매치**: 피부 타입별 전문 카운슬러 배치가 안 되어 상담 품질 불균일
4. **재방문율 저조**: 첫 방문 체험은 긍정적이나, 재방문으로 이어지지 않는 구조

### 도입 목표

- 카운슬링 존 효율 극대화 (상담 전환율 +40% 이상)
- 테스터 존 동선 분산으로 체험 만족도 향상
- 데이터 기반 상담원 배치
- 재방문율 30일 기준 25% 이상 달성

## 4. NeuralTwin 활용 기능 상세

### 4.1 카운슬링존 실시간 혼잡도 모니터링

**적용 내용**:
- 카운슬링 존(Z5) 전용 센서로 실시간 인원 감지
- 혼잡도 3단계 표시: 여유(초록) / 보통(노랑) / 혼잡(빨강)
- 혼잡 시 자동 대기 번호 발급 + 예상 대기 시간 안내
- 대기 중 테스터 존(Z4) 또는 한정판 존(Z6) 방문 유도 알림

**결과**:
- 카운슬링 대기 포기율: 35% → 12% (-66%)
- 카운슬링 체류시간: 8분 → 15분 (+87%) — 충분한 상담으로 전환율 상승
- 상담 후 즉시 구매율: 18% → 29% (+61%)

### 4.2 AI 기반 상담원 배치 추천

**적용 내용**:
- 시간대별/요일별 방문자 프로필 패턴 분석 (연령대, 체류 패턴)
- 카운슬러별 강점 매핑 (스킨케어 전문, 메이크업 전문, VIP 응대)
- AI가 시간대별 최적 카운슬러 조합 추천
- 피크 타임 자동 알림으로 카운슬러 증원 트리거

**결과**:
- 상담 만족도: 3.8/5 → 4.5/5 (+18%)
- 1인당 상담 건수: 12건/일 → 15건/일 (+25%)
- 상담 → 구매 전환율: 22% → 34% (+55%)

### 4.3 테스터 존 동선 최적화

**적용 내용**:
- 테스터 존(Z4) 내 12개 테스터 스테이션의 체류 패턴 분석
- 인기 테스터 3곳에 80% 고객 집중 현상 발견
- AI 추천: 테스터 배치 재구성 + 동선 유도 바닥 사이니지 + "숨은 인기 제품" POP
- 비인기 테스터에 향(Scent) 디퓨저 연계 체험 동선 설계

**결과**:
- 테스터 존 활용률: 55% → 80% (+45%)
- 테스터 체험 후 구매 전환: 8% → 15% (+88%)
- 테스터 존 평균 체류시간: 5분 → 9분 (+80%)
- 체험 제품 수 (1인당 평균): 2.1개 → 3.8개 (+81%)

## 5. ROI 상세 계산

### 비용 구조

| 항목 | 월 비용 | 비고 |
|------|---------|------|
| NeuralTwin Professional Tier | ₩490,000 | 기본 구독료 |
| 센서 하드웨어 (7대) | ₩97,000 | 초기 ₩580,000 ÷ 6개월 상각 |
| 설치 및 캘리브레이션 | ₩50,000 | 초기 ₩300,000 ÷ 6개월 상각 |
| **월 총 비용** | **₩637,000** | |

> **참고**: 센서 하드웨어 및 설치비는 초기 비용을 6개월 상각 기준으로 월할 계산했습니다. 7개월차부터 월 비용은 ₩490,000으로 감소합니다.

### 수익 증가

| 항목 | 월 증가액 | 산출 근거 |
|------|----------|-----------|
| 매출 증가분 | +₩70,000,000 | ₩280M - ₩210M |
| 추정 영업이익률 | 20% | 뷰티 리테일 업계 평균 |
| **월 영업이익 증가분** | **₩14,000,000** | |

### ROI 계산

```
월 순이익 증가 = ₩14,000,000 (영업이익 증가)
월 총 비용     = ₩637,000

ROI = (₩14,000,000 - ₩637,000) / ₩637,000 × 100
    = ₩13,363,000 / ₩637,000 × 100
    ≈ 2,097% (약 21x)

매출 기준 ROI = ₩70,000,000 / ₩490,000
             ≈ 143x (매출 증가 대비 구독료)
```

### 손익분기점

```
월 구독료 ₩490,000 ÷ 영업이익률 20% = 월매출 ₩2,450,000 증가 시 BEP
실제 월매출 증가 ₩70,000,000 → 도입 첫 달부터 BEP 달성
```

## 8. 뷰티 업종 특화 인사이트

### 뷰티 리테일의 핵심 전환 포인트

NeuralTwin 데이터 분석으로 뷰티 플래그십 매장의 구매 전환에 가장 큰 영향을 미치는 요인 3가지를 식별했습니다:

| 순위 | 전환 요인 | 기여도 | NeuralTwin 대응 |
|------|----------|--------|----------------|
| 1 | 카운슬링 체류시간 15분 이상 | 38% | 혼잡도 관리로 충분한 상담 시간 확보 |
| 2 | 테스터 체험 3개 제품 이상 | 28% | 동선 최적화로 체험 수 증가 |
| 3 | 재방문 2회 이상 | 22% | 체험 데이터 기반 개인화 리마인더 |

### 성수동 상권 특성 반영

- **MZ세대 비중 높음** (20~35세 72%) → 디지털 대기 시스템 수용도 높음
- **SNS 공유 행동** → 체험존 체류시간 증가가 자연스러운 바이럴 마케팅 효과
- **주말 방문 집중** (주말 방문자 = 평일 × 2.3) → 주말 특화 인력 배치 필수

## 9. 방법론 및 데이터 신뢰성

### 측정 방법론
- **방문자 수**: WiFi Probe MAC 주소 기반 고유 방문자 카운팅 (MAC 랜덤화 보정 알고리즘 적용)
- **상담 전환율**: 카운슬링 존 진입 후 30분 이내 POS 거래 발생 비율
- **카운슬링 체류시간**: 카운슬링 존(Z5) 진입 ~ 이탈 시점 차이
- **테스터 존 효율**: (실제 사용 테스터 수 / 총 테스터 수) × (테스터당 평균 체험자 수 / 목표 체험자 수)
- **재방문율**: 30일 내 동일 MAC 해시 재감지 비율

### 외부 변수 통제
- 분석 기간 중 브랜드 콜라보/팝업 기간은 별도 표기
- 계절 변동성은 전년 동기 대비로 보정
- 성수동 상권 전체 방문자 추이 참조 (서울시 유동인구 데이터)

### 데이터 수집 준수 사항
- 개인정보보호법 준수: MAC 주소 비가역 해싱 처리
- 매장 내 데이터 수집 고지 안내문 게시
- 수집 데이터 90일 자동 파기 정책 적용
