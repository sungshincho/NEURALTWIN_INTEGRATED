## 2. 도입 배경 및 과제

### 기존 문제점

1. **시식 체험존 과밀**: 주말 시식존(Z4) 점유율 130% 초과 — 소방법 기준 초과 우려, 고객 불만
2. **동선 편중**: 전체 방문자의 62%가 Z1→Z4→Z8 직행, 나머지 존 방문율 저조
3. **크로스셀링 부재**: 시식 후 해당 브랜드만 구매, 인접 존 연계 구매 비율 12%에 불과
4. **시간대별 프로모션 비효율**: 종일 동일 시식 운영, 피크/비피크 타임 차별화 없음

### 도입 목표

- 시식존 혼잡도 정상화 (점유율 90% 이하)
- 인접 존 동선 유입 +30% 이상
- 크로스셀링 비율 20% 이상 달성
- 월매출 +10% 이상 성장

## 4. NeuralTwin 활용 기능 상세

### 4.1 실시간 혼잡도 대시보드

**적용 내용**:
- 8개 존 전체에 WiFi Probe 센서 10대 설치, 실시간 존별 인원 모니터링
- 관리자용 대시보드: 존별 점유율 실시간 표시 (5초 갱신)
- 시식존(Z4) 점유율 90% 초과 시 자동 알림 → 스태프 동선 분산 유도 개시
- 고객용 디지털 사이니지: "지금 여유로운 존" 실시간 안내

**결과**:
- 시식존 피크 점유율: 130% → 85% (혼잡 해소)
- 시식존 관련 고객 불만: 월 23건 → 월 4건 (-83%)
- 소방법 기준 초과 발생: 주 3~4회 → 0회

### 4.2 시식존 분산 배치 AI 추천

**적용 내용**:
- 시식 코너 8개소의 시간대별 인기도 분석
- AI 추천: 인기 시식을 시간대별로 분산 운영 (A시식 오전 / B시식 오후)
- 비인기 존(Z7 와인존) 근처에 와인 페어링 시식 코너 신설
- Z2(과일) → Z4(시식) → Z5(델리) 연결 동선 유도

**결과**:
- 시식 코너 간 방문 균형도: 편차 45% → 편차 18% (균등화)
- Z7 와인존 방문율: +48% (와인 시식 페어링 효과)
- 시식 → 인접 존 이동율: 28% → 51% (+82%)

### 4.3 시간대별 프로모션 최적화

**적용 내용**:
- 시간대별 방문자 프로필 패턴 분석:
  - 오전(10:30~12:00): 주부 중심, 과일/채소 수요
  - 점심(12:00~14:00): 직장인 점심, 델리/HMR 수요
  - 오후(14:00~17:00): 가족 단위, 시식/체험 수요
  - 저녁(17:00~20:00): 퇴근 후 장보기, 정육/수산 수요
- AI 추천 기반 시간대별 프로모션 차등 운영
- 디지털 사이니지 연동: 시간대에 따라 자동 프로모션 콘텐츠 변경

**결과**:
- 시간대별 프로모션 응답률: 평균 8% → 14% (+75%)
- 비피크 타임(14:00~17:00) 매출: +22%
- 저녁 타임(17:00~20:00) 정육/수산 매출: +18%

## 5. ROI 상세 계산

### 비용 구조

| 항목 | 월 비용 | 비고 |
|------|---------|------|
| NeuralTwin Enterprise Tier | ₩1,490,000 | 기본 구독료 |
| 센서 하드웨어 (10대) | ₩140,000 | 초기 ₩840,000 ÷ 6개월 상각 |
| 설치 및 캘리브레이션 | ₩83,000 | 초기 ₩500,000 ÷ 6개월 상각 |
| 디지털 사이니지 연동 | ₩100,000 | API 연동 유지비 |
| **월 총 비용** | **₩1,813,000** | |

> **참고**: 센서 하드웨어 및 설치비는 초기 비용을 6개월 상각 기준으로 월할 계산했습니다. 7개월차부터 월 비용은 ₩1,590,000으로 감소합니다.

### 수익 증가

| 항목 | 월 증가액 | 산출 근거 |
|------|----------|-----------|
| 매출 증가분 | +₩160,000,000 | ₩1,580M - ₩1,420M |
| 추정 영업이익률 | 8% | 백화점 식품관 업계 평균 |
| **월 영업이익 증가분** | **₩12,800,000** | |

### ROI 계산

```
월 순이익 증가 = ₩12,800,000 (영업이익 증가)
월 총 비용     = ₩1,813,000

ROI = (₩12,800,000 - ₩1,813,000) / ₩1,813,000 × 100
    = ₩10,987,000 / ₩1,813,000 × 100
    ≈ 606% (약 6x)

매출 기준 ROI = ₩160,000,000 / ₩1,490,000
             ≈ 107x (매출 증가 대비 구독료)
```

### 손익분기점

```
월 구독료 ₩1,490,000 ÷ 영업이익률 8% = 월매출 ₩18,625,000 증가 시 BEP
실제 월매출 증가 ₩160,000,000 → 도입 첫 달부터 BEP 달성
```

## 9. 방법론 및 데이터 신뢰성

### 측정 방법론
- **방문자 수**: WiFi Probe MAC 주소 기반 고유 방문자 카운팅 (MAC 랜덤화 보정 알고리즘 적용)
- **시식존 점유율**: (실시간 감지 인원 / 소방법 기준 수용 인원) × 100
- **크로스셀링 비율**: 2개 이상 존에서 구매 발생한 고객 / 전체 구매 고객 × 100
- **체류시간**: 최초 감지 ~ 최종 감지 시간 차이 (10분 이상 체류자만 유효, 식품관 특성 반영)
- **인접 존 유입**: 시식존(Z4) 체류 후 10분 이내 인접 존(Z3, Z5, Z6) 감지 비율

### 외부 변수 통제
- 백화점 전관 세일 기간(더현대 세일 등)은 별도 표기
- 계절 변동성은 전년 동기 대비로 보정
- 명절(설/추석) 기간 데이터는 일반 기간과 분리 분석
- 백화점 전체 방문자 추이와 연동하여 식품관 고유 효과 분리

### 데이터 수집 준수 사항
- 개인정보보호법 준수: MAC 주소 비가역 해싱 처리
- 백화점 고객 안내 게시판 및 식품관 입구 데이터 수집 고지
- 수집 데이터 90일 자동 파기 정책 적용
- 백화점 본사 개인정보 보호 정책과 연동
