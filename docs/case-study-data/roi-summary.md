### ROI 산출 기준

```
매출 기준 ROI = 월매출 증가분 / NeuralTwin 월 구독료
영업이익 기준 ROI = (월 영업이익 증가 - 월 총 비용) / 월 총 비용 × 100
```

> **주의**: 매출 기준 ROI는 NeuralTwin 구독료 대비 매출 증가의 배율을 나타내며, 투자 회수 판단에는 영업이익 기준 ROI를 사용하는 것이 적절합니다.

## 8. 방법론 공통 사항

### 데이터 수집 방식
- WiFi Probe 기반 MAC 주소 감지 → 비가역 해싱 처리
- MAC 랜덤화 보정 알고리즘 적용 (정확도 92~95%)
- 5분(패션/뷰티) ~ 10분(식품관) 이상 체류자만 유효 방문자로 카운팅

### 매출 데이터 연동
- POS 시스템 API 연동 또는 일 단위 수동 입력
- 전환율 = POS 거래 건수 / 고유 방문자 수 × 100

### 외부 변수 통제
- 대규모 프로모션/세일 기간 별도 표기
- 계절 변동성은 전년 동기 대비 보정
- 외부 이벤트(공사, 축제 등) 발생 시 해당 일자 주석 처리

### 개인정보 보호
- 개인정보보호법 준수: MAC 주소 비가역 해싱
- 매장 내 데이터 수집 고지 안내문 게시 의무
- 수집 데이터 90일 자동 파기 정책
